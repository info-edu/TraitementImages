Deuxième sous section nb¶

Ceci est une cellule de texte écrite en markdown. On peut double cliquer dessus pour voir le code markdown qui y correspond et le changer. Quand on l’exécute, le code markdown se transforme en une jolie mise en forme. Ce langage nous permettra d’intégrer facilement différentes mises en forme avec nos programmes. On pourra ainsi entre-autres intégrer: + des titres, sous titres… avec l’opérateur # + des puces comme celles-ci + des liens comme celui-ci + des images, avec une commande similaire à celle pour les liens, à la différence qu’on mettra un ! avant.

Particularité d’un notebook¶

Attention, dans jupyter notebook, une fois qu’une cellule de code a été exécutée, si des variables ont étées définies dans cette cellule, alors elles le sont pour la page entière.

Essayez: définissez une variable x dans une cellule, exéctuez là, puis exécutez une cellule avec print(x) à l’intérieur dans une cellule au dessus de celle précédente. Normalement vous verrez que x peut maintenant être utilisé dans n’importe quelle cellule.

[3]:

print(x)

test

[2]:

x = "test"

Raccourcis clavier¶

Sur jupyter, deux raccourcis clavier sont très utiles: + CMD + S pour sauvegarder votre fichier + SHIFT + ENTER pour executer une cellule
